University Game
During my time at University I took a games module, where we created a game based off of an early, basic 2D shooter. In this project I had to choose my own theme, where I went for a cartoon/anime theme. So this game is similar and based off of Space Invaders, where the player will constantly be shooting, while being able to move in all directions. Despite this, the player always travels to the right of the screen, so the shots all go to the right. With 3 lives, you must defeat the boss to win the game! This was an early projectand and looking back on it now, there are things I might have changed or shortcuts that would have helped me achieve my goal quicker. Click here to see this GitHub repository!
